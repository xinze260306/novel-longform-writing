# 长文小说肌肉记忆写作法

> **方法来源**：道家"守一"心法 + 工匠精神
> **适用范围**：100-200章以上的长篇小说
> **核心原则**：慢就是快，稳扎稳打，闭环验证

---

## 核心理念

### 道：守一而止
- 一章一章写，不跳写、不并行
- 每章必须经过完整11步闭环
- 数量执念是心魔，质量第一

### 法：实事求是
- 每章写完后必须朗读验证
- 关键数据（时间/数字）必须复查
- 人物言行必须符合人设

### 术：肌肉记忆
- 严格按照11步流程执行
- 不跳过任何步骤，不凭感觉写作
- 重复形成肌肉记忆，降低认知负担

### 器：工具辅助
- 使用 edge-tts 生成配音验证
- 使用 ffmpeg 合成视频预览
- 使用文件系统管理章节

## 11步写作闭环（每章必须执行）

```
1. 朗读原文 → 2. 标注扩写点 → 3. 人物心理扩展
    ↓
4. 场景氛围扩展 ← 5. 对话艺术扩展 ← 6. 战斗艺术扩展
    ↓
7. 第N次朗读验证 → 8. 视频预览生成 → 9. 交叉验证
    ↓
10. 第N+1次朗读定稿 → 11. 记录到扩写统计表
```

### 详细步骤

#### 第1步：朗读原文
- 朗读当前章节原文3-5分钟
- 感受节奏和语气
- 标记读起来生硬的地方

#### 第2步：标注扩写点
- 确定需要扩展的段落
- 标记扩展类型（心理/场景/对话/战斗）

#### 第3步：人物心理扩展
- 增加内心独白和情感层次
- 展现人物成长弧光

#### 第4步：场景氛围扩展
- 五感描写（视觉、听觉、嗅觉、触觉、味觉）
- 环境细节和氛围渲染

#### 第5步：对话艺术扩展
- 符合人物身份的说话方式
- 潜台词和言外之意
- 避免"琼瑶式"重复

#### 第6步：战斗/冲突艺术扩展
- 战斗不仅是动作，更是意志较量
- 招式名称+心理活动+环境影响

#### 第7步：第N次朗读验证
- 扩写后朗读，检查流畅度
- 标记仍有问题的地方

#### 第8步：视频预览生成（可选）
- 使用 edge-tts 生成配音
- 合成视频预览效果
- 通过听觉发现阅读时忽略的问题

#### 第9步：交叉验证
- 对照人物设定表检查言行一致性
- 对照时间线检查逻辑合理性
- 对照前文检查伏笔呼应

#### 第10步：第N+1次朗读定稿
- 最终朗读确认
- 确保流畅自然

#### 第11步：记录到扩写统计表
- 记录本章扩写数据
- 形成可追溯的历史记录

## 工作流程

### 阶段1：大纲设计
1. 确定总章节数（建议100-200章）
2. 设计详细大纲（每章500字以上概要）
3. 建立人物设定表
4. 建立时间线/地图设定

### 阶段2：章节扩写
严格按照11步闭环，一章一章完成：
- 不跳写、不并行
- 每章必须经过完整闭环
- 完成一章再开始下一章

### 阶段3：质量检查
- 使用"三版本朗读验证法"：
  - 版本1：作者自己朗读
  - 版本2：TTS机器朗读
  - 版本3：休息后重新朗读

## 防心魔机制

### 心魔症状识别
- [ ] 急于求成，想同时写多章
- [ ] 跳过验证步骤
- [ ] 执念于"完成200章"的目标
- [ ] 忽略大纲凭感觉写作

### 心魔防治口诀
```
致虚极，守静笃。
一章一章，稳扎稳打。
慢就是快，少就是多。
质量第一，数量随缘。
```
