# 长文小说写作方法论 v2.0

> 基于道家"守一"心法 + 工匠精神

---

## 一、核心理念

### 1.1 道：守一而止

**"守一"出自《道德经》**：
> "圣人抱一为天下式。"

**在写作中的应用**：
- 一次只写一章
- 一章只有一个核心目标
- 不跳写、不并行、不贪多

**常见心魔**：
- ❌ "我先写个框架，再回头填内容"
- ❌ "这几章有关联，我一起写"
- ❌ "我先写高潮部分，再补铺垫"

**正确做法**：
- ✅ 严格按照大纲顺序，一章一章写
- ✅ 每章必须经过完整闭环才能开始下一章
- ✅ 遇到关联内容，标记"待第X章呼应"，继续当前章

### 1.2 法：实事求是

**三验证原则**：
1. **朗读验证**：写完必须朗读，不通顺的地方修改
2. **数据验证**：时间、数字、人名必须复查
3. **人设验证**：人物言行必须符合设定

**实事求是的具体做法**：
- 不凭记忆写作，对照大纲和设定表
- 不确定的内容查证后再写
- 错了就改，不找借口

### 1.3 术：肌肉记忆

**什么是肌肉记忆？**
- 不需要思考，身体自动执行
- 重复足够多的次数后形成
- 降低认知负担，提高执行效率

**如何形成写作肌肉记忆？**
1. 严格按照11步流程执行100次以上
2. 不跳过步骤，不偷工减料
3. 重复到成为本能反应

### 1.4 器：工具辅助

**推荐工具链**：
- **配音**：edge-tts（验证朗读流畅度）
- **视频**：ffmpeg（合成预览视频）
- **版本控制**：git（追踪修改历史）
- **统计**：Markdown表格（记录扩写数据）

## 二、11步写作闭环详解

### 第1步：朗读原文（5-10分钟）

**目的**：建立语感，发现问题

**做法**：
1. 大声朗读当前章节原文
2. 感受节奏是否流畅
3. 标记读起来拗口的地方

**检查点**：
- [ ] 是否有长句需要断开？
- [ ] 是否有重复词语？
- [ ] 对话是否自然？

### 第2步：标注扩写点（5分钟）

**扩写类型识别**：

| 类型 | 识别特征 | 扩写方向 |
|------|----------|----------|
| 心理 | "他很生气" | 展开内心活动 |
| 场景 | "走进房间" | 五感细节描写 |
| 对话 | "他说" | 增加语气、动作、潜台词 |
| 战斗 | "打了一拳" | 招式+心理+环境 |
| 转折 | "突然" | 铺垫+层次递进 |

**标注方法**：
```markdown
原文：[他很生气]
扩写类型：心理
扩写方向：展现从压抑到爆发的过程
```

### 第3步：人物心理扩展

**心理描写层次**：
1. **表层**：直接情绪（生气、开心）
2. **中层**：原因分析（为什么生气）
3. **深层**：价值观冲突（这违背了我的什么原则）
4. **转变**：心理变化（从A状态到B状态）

**示例**：
```
原文：他很生气。

扩写后：
张三握紧拳头，指节发白。（表层-动作表现）

他盯着李四，脑子里嗡嗡作响。（中层-生理反应）

"这混蛋，明明答应过的事..."（中层-内心独白）

但更让张三愤怒的不是背叛本身，
而是自己竟然又一次相信了别人。（深层-自我反思）

他深吸一口气，慢慢松开拳头。（转变-克制）
"好，很好。"他平静地说，声音却冷得像冰。
（转变-情绪压抑）
```

### 第4步：场景氛围扩展

**五感描写法**：

| 感官 | 描写维度 | 示例 |
|------|----------|------|
| 视觉 | 颜色、光影、形状 | "夕阳将天空染成血色" |
| 听觉 | 声音、音量、音质 | "远处传来乌鸦的叫声" |
| 嗅觉 | 气味、香臭 | "空气中弥漫着血腥气" |
| 触觉 | 温度、质地 | "夜风刺骨，像刀子一样" |
| 味觉 | 味道 | "嘴里泛起铁锈般的血腥味" |

**氛围营造技巧**：
- 用环境衬托心情（以景写情）
- 用细节暗示局势（以小见大）
- 用变化推动情绪（从静到动）

### 第5步：对话艺术扩展

**对话三要素**：
1. **说什么**（内容）
2. **怎么说**（语气、方式）
3. **为什么这么说**（动机、潜台词）

**避免琼瑶式对话**：
- ❌ "你真的要这样对我吗？"
- ❌ "你怎么可以这样？"
- ❌ "我恨你！"

**正确写法**：
- ✅ 用行动代替情绪词
- ✅ 用潜台词代替直白表达
- ✅ 用沉默和停顿增加张力

**示例**：
```
琼瑶式：
"你怎么可以背叛我？"她哭着说。

改进后：
她看着他，很久，久到眼睛发酸。
"那笔钱，"她声音很轻，"是你拿的？"

他没有回答。

"好。"她点点头，转身走向门口，
"钥匙我放桌上了。"

门关上，咔哒一声。
```

### 第6步：战斗/冲突艺术扩展

**战斗描写五层次**：
1. **招式**：具体的动作和技术
2. **力量**：力度、速度、角度
3. **心理**：战斗者的内心活动
4. **环境**：战斗场景的影响
5. **意义**：这场战斗的象征意义

**战斗节奏控制**：
- 快-慢-快：快节奏→慢镜头→高潮
- 动-静-动：激烈→停顿→更激烈
- 攻-守-攻：攻击→防守→反击

### 第7步：第N次朗读验证

**验证重点**：
- 扩写后的内容是否流畅？
- 新增内容是否破坏了原有节奏？
- 字数是否符合预期？

### 第8步：视频预览生成（可选）

**为什么需要视频预览？**
- 听觉发现阅读时忽略的问题
- 模拟读者体验
- 增加仪式感，标志着一章的完成

**工具链**：
```bash
# 1. 生成配音
edge-tts --voice zh-CN-XiaoxiaoNeural --text "$(cat chapter.md)" --write-media chapter.mp3

# 2. 合成视频
ffmpeg -i chapter.mp3 -filter_complex "..." -shortest chapter.mp4
```

### 第9步：交叉验证

**验证清单**：

| 验证项 | 检查内容 | 工具 |
|--------|----------|------|
| 人设一致性 | 言行是否符合设定 | characters/ |
| 时间线 | 事件顺序是否正确 | timeline.md |
| 地理 | 场景转换是否合理 | map.md |
| 伏笔 | 前文伏笔是否呼应 | 搜索功能 |
| 数据 | 数字是否一致 | 全局搜索 |

### 第10步：第N+1次朗读定稿

**定稿标准**：
- 朗读流畅，无拗口之处
- 逻辑自洽，无矛盾之处
- 情感真挚，无虚假之处
- 细节丰富，无遗漏之处

### 第11步：记录到扩写统计表

**统计表模板**：
```markdown
| 章节 | 原文(字) | 扩写后(字) | 扩写比 | 主要扩写点 | 耗时 | 状态 |
|------|----------|------------|--------|------------|------|------|
| 001 | 500 | 2500 | 5x | 心理、场景 | 2h | ✅ |
| 002 | 600 | 2800 | 4.7x | 对话、战斗 | 2.5h | ✅ |
```

## 三、防心魔机制

### 心魔症状识别

| 症状 | 表现 | 后果 |
|------|------|------|
| 急于求成 | 想同时写多章 | 质量下降，返工增加 |
| 跳过验证 | 不朗读、不检查 | 错漏百出，读者流失 |
| 数量执念 | 只关注"完成200章" | 忽视质量，走火入魔 |
| 感觉写作 | 不按大纲凭感觉 | 偏离主线，结构崩坏 |

### 心魔防治口诀

```
致虚极，守静笃。
五色令人目盲，五音令人耳聋。

一章一章，稳扎稳打。
慢就是快，少就是多。
质量第一，数量随缘。

源头第一，大纲为准。
实事求是，闭环验证。
```

### 走火入魔急救法

**如果发现自己出现心魔症状**：
1. 立即停止写作
2. 深呼吸三次
3. 默念口诀
4. 向师尊/同伴汇报
5. 等待指示，不擅自继续

│   ├── map.md

## 五、常见问题

### Q1：扩写后字数太多怎么办？

A：字数不是目标，质量才是。如果扩写后内容充实、节奏紧凑，字数多不是问题。如果感觉冗余，回到第7步朗读验证，删减冗余部分。

### Q2：遇到写不下去的情况怎么办？

A：
1. 检查是否跳过了某个步骤
2. 返回大纲，确认本章目标
3. 使用"最小可行章节"策略：先写出基本框架，再逐步扩展
4. 如果实在写不下去，标记"待补充"，开始下一章（这是例外，不是常态）

### Q3：如何保持长期写作动力？

A：
1. **小闭环**：每完成一章就是一次胜利
2. **可视化**：用统计表看到进度
3. **外部反馈**：定期给信任的人看稿
4. **仪式感**：配音、视频预览作为完成的标志
5. **休息**：遵循"十轮后休息"原则

### Q4：如何处理读者的反馈？

A：
1. 感谢所有反馈
2. 区分"事实"（有错别字）和"观点"（我不喜欢这个角色）
3. 事实类立即修改
4. 观点类记录后统一考虑
5. 不因为个别意见改变整体方向

## 六、进阶技巧

### 6.1 伏笔铺设

**三层伏笔法**：
1. **表层伏笔**：读者能猜到会发生什么
2. **中层伏笔**：读者以为知道会发生什么，但实际不同
3. **深层伏笔**：读者完全没意识到，揭晓时恍然大悟

### 6.2 节奏控制

**呼吸节奏**：
- 紧张场景后要有放松（让读者喘气）
- 信息密集后要有消化（让读者理解）
- 连续高潮之间要有铺垫（让读者期待）

### 6.3 多线叙事

**主线-支线处理**：
- 主线：每章必须推进
- 支线：可以隔几章再提
- 交叉：在关键节点让主线和支线交汇
